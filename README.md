# Submission Instructions

1. Clone the repository.
2. Rename the `<rollno>` folder with your actual roll number.
3. Complete all the tasks stepwise and carefully follow the instructions provided.
4. Upload your submissions in their respective folders:

   * MNIST-related work inside `MNIST_DigitClassification/`
   * SLM-related work inside `SLM/`
5. Push only the required Jupyter Notebook (`.ipynb`) or files mentioned in the task instructions.
6. Commit and push your changes properly.

Example:

```bash id="x0c7tw"
git add .
git commit -m "Added task submission"
git push
```
